## なぜこのMOCがあるか

技術記事・公式ドキュメント・書籍・動画など多様な情報源から学ぶが、断片化すると後で活かしづらい。「読んだもの」の入口を一カ所にまとめ、関連する学習・プロダクトのMOCに必ずリンクさせる運用にする。CLAUDE.md の毎朝ルーチン（Claude Code チェンジログ、Anthropic公式ニュース、Hacker News、OpenAI / Codex / Cursor / GitHub Copilot、Vercel / Supabase / セキュリティニュース）の入力もここに集約。

## ノート運用ルール

- フォーマットは `#reading` タグを必ず付与
- 1記事/1書籍 = 1ノート（または「同テーマで読み比べ」レビューを1ノートにまとめる）
- 必ず以下を含める：
  - 出典URL / ISBN
  - 読んだ日付
  - 3行サマリ
  - 自分の言葉での要点
  - 関連MOCへのリンク（ここ含む）
- 既存ノートと矛盾する内容を読んだら **#矛盾** タグでフラグ
